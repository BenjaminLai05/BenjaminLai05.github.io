NBA Algorithm Predictor

This system collects labeled examples by web scraping Basketball-Reference to extract game results and player statistics from automation with Selenium. The preprocessing pipeline removes, cleans, parses, filters and merges multiple sources of wrangled datasets in order to backtest predictions by using a forward game-by-game rating system which evaluates accuracy by comparing predicted winners to actual results.
